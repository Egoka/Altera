### Основные технологии:
- **Nuxt 4** - фреймворк
- **Vue 3.5.17** - UI библиотека
- **Tailwind CSS v4** - стилизация
- **FishtVue** - компонентная библиотека
- **TypeScript** - типизация

### Модули Nuxt:
- **@nuxt/eslint** - линтер
- **@nuxt/fonts** - шрифты
- **@nuxt/icon** - иконки
- **@nuxt/image** - оптимизация изображений
- **@nuxt/test-utils** - тестирование

- **@pinia/nuxt** - управление состоянием
