# Руководства и стандарты

Эта папка содержит руководства и стандарты для разработки проекта Altera.

## 📄 Файлы

### 📄 authentication_guide.md
Руководство по тестированию Magic Link авторизации:
- Пошаговое руководство по тестированию авторизации
- Примеры GraphQL запросов
- Настройка заголовков для аутентифицированных запросов
- Обработка ошибок авторизации

### 📄 admin_query_standards.md
Стандарты для админ-запросов:
- Единообразие API для админ-панели
- Стандартные структуры Input/Output типов
- Принципы производительности и безопасности
- Примеры реализации сложных таблиц с фильтрацией

### 📄 article_workflow.md
Рабочий процесс статей:
- Жизненный цикл статьи (черновик → ревью → публикация)
- Роли пользователей в процессе
- Автоматизация рабочих процессов
- Интеграция с админ-панелью

## 🎯 Назначение

Эти документы предназначены для:
- Новых разработчиков, присоединяющихся к проекту
- Понимания стандартов и лучших практик
- Обеспечения консистентности в разработке
- Обучения работе с различными компонентами системы
