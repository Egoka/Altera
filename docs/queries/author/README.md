# Author GraphQL Queries

Эта папка содержит GraphQL запросы для работы с авторами и их страницами.

## Файлы

### 📄 author.graphql
Комплексные запросы для страниц авторов:

#### Основные запросы:
- **GetAuthorPage** - комплексный запрос для получения всей информации для страницы автора
- **GetAuthorInfo** - получение только информации об авторе
- **GetAuthorArticles** - получение статей автора с пагинацией
- **GetAuthorStats** - получение статистики автора

#### Возвращаемые данные:
- Публичная информация об авторе (имя, био, фото, социальные ссылки)
- Список опубликованных статей с пагинацией
- Статистика (общее количество статей, статьи за месяц, популярные теги)

## Особенности запросов авторов

- Запросы оптимизированы для публичных страниц
- Включают только опубликованные статьи
- Поддерживают пагинацию для больших списков статей
- Возвращают статистику для отображения на странице автора
