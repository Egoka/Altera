# Admin GraphQL Queries

Эта папка содержит GraphQL запросы для админ-панели. Все запросы требуют аутентификации с правами администратора.

## Файлы

### 📄 article.graphql
Запросы для управления статьями в админ-панели:
- Получение списка статей с пагинацией и фильтрацией
- Фильтрация по статусу (DRAFT, REVIEW, PUBLISHED, ARCHIVED)
- Поиск по содержимому
- Сортировка по различным полям
- Получение детальной информации о статье

### 📄 user.graphql
Запросы для управления пользователями:
- Получение списка пользователей
- Фильтрация по ролям
- Поиск пользователей
- Получение профиля пользователя

### 📄 tag.graphql
Запросы для управления тегами:
- Получение списка тегов
- Создание и редактирование тегов
- Управление связями тегов со статьями

### 📄 content_type.graphql
Запросы для управления типами контента:
- Получение списка типов контента
- Создание и редактирование типов
- Настройка полей для типов контента

## Особенности админских запросов

- Все запросы включают расширенную информацию (статусы, даты создания/обновления)
- Поддерживается сложная фильтрация и сортировка
- Запросы оптимизированы для админ-интерфейса
- Включают пагинацию для больших списков
