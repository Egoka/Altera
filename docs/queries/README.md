# GraphQL Query Documentation

Эта папка содержит тестовые GraphQL запросы, организованные по категориям для удобства разработки и тестирования.

## Структура папок

### 📁 admin/
Содержит запросы для админ-панели:
- `article.graphql` - управление статьями
- `user.graphql` - управление пользователями  
- `tag.graphql` - управление тегами
- `content_type.graphql` - управление типами контента

### 📁 author/
Содержит запросы связанные с авторами:
- `author.graphql` - запросы для страниц авторов

### 📁 common/
Содержит общие запросы для публичной части:
- `main.graphql` - основные запросы (главная страница, профиль)
- `article.graphql` - запросы статей
- `content_type.graphql` - запросы типов контента
- `tag.graphql` - запросы тегов

## Использование

Эти файлы предназначены для:
- Тестирования GraphQL API
- Документирования доступных запросов
- Примеров использования для разработчиков
- Проверки кэширования и производительности

## Примечания

- Все запросы протестированы и работают с текущей схемой
- Запросы включают примеры с различными параметрами фильтрации и сортировки
- Для админских запросов требуется аутентификация с правами администратора
