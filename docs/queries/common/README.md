# Common GraphQL Queries

Эта папка содержит общие GraphQL запросы для публичной части приложения.

## Файлы

### 📄 main.graphql
Основные запросы для главных страниц:
- **getHomePageArticles** - запрос для главной страницы (featured, latest, popular статьи)
- **getMyProfile** - запрос профиля пользователя
- **getArticle** - запрос отдельной статьи

### 📄 article.graphql
Запросы для работы со статьями:
- Получение списка статей с фильтрацией
- Поиск по статьям
- Получение статей по тегам и типам контента

### 📄 content_type.graphql
Запросы для работы с типами контента:
- Получение списка типов контента
- Получение статей по типу контента

### 📄 tag.graphql
Запросы для работы с тегами:
- Получение списка тегов
- Получение статей по тегам
- Популярные теги

## Особенности общих запросов

- Запросы оптимизированы для публичных страниц
- Включают только опубликованный контент
- Поддерживают базовую фильтрацию и поиск
- Возвращают данные, необходимые для отображения в UI
