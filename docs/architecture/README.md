# Архитектурная документация

Эта папка содержит документацию по архитектуре системы Altera.

## 📄 Файлы

### 📄 database_architecture.md
Архитектура базы данных:
- Диаграмма Entity-Relationship (E-R)
- Описание таблиц и их связей
- Примеры SQL запросов
- Принципы проектирования БД

#### Основные сущности:
- **User** - пользователи системы
- **Article** - статьи и контент
- **ContentType** - типы контента
- **SectionTag** - теги и разделы

#### Связи:
- User → Article (один ко многим)
- Article → ContentType (многие к одному)
- Article ↔ SectionTag (многие ко многим)

## 🎯 Назначение

Эта документация предназначена для:

- Планирования изменений в схеме БД
- Обучения новых разработчиков
- Обеспечения консистентности в разработке
