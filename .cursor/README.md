# Cursor AI Configuration for Altera Project

Эта папка содержит настройки AI-ассистента Cursor для проекта Altera.

## Структура файлов

### `settings.json`
Основные настройки AI-ассистента:
- **model**: Claude 3.5 Sonnet (оптимальный баланс скорости и качества)
- **temperature**: 0.1 (низкая креативность для консистентного кода)
- **contextWindow**: 200000 токенов (большой контекст для понимания проекта)
- **autoComplete**: включено для быстрого кодирования

### `rules`
Правила и конвенции проекта:
- Архитектура Nuxt.js + Vue 3 + Prisma
- TypeScript везде
- Конвенции именования
- Best practices для безопасности и производительности

### `prompts`
Готовые промпты для частых задач:
- Создание компонентов Vue
- Разработка API endpoints
- Работа с MDC форматом
- Тестирование и документация

### `snippets.json`
Пользовательские сниппеты:
- `vue-comp` - Vue компонент с TypeScript
- `nuxt-page` - Nuxt страница с SEO
- `api-route` - API маршрут с валидацией
- `composable` - Vue composable
- `prisma-model` - Prisma модель
- `ts-interface` - TypeScript интерфейс
- `zod-schema` - Zod схема валидации
- `test-file` - Файл тестов

## Как использовать

### 1. Сниппеты
Введите префикс сниппета (например, `vue-comp`) и нажмите Tab.

### 2. Промпты
Скопируйте нужный промпт из файла `prompts` и вставьте в чат Cursor.

### 3. Правила
AI автоматически следует правилам из файла `rules` при генерации кода.

## Настройки для команды

Эти файлы можно коммитить в Git для обеспечения консистентности настроек AI-ассистента в команде. Каждый разработчик будет получать одинаковые подсказки и следовать одним правилам.

## Кастомизация

Вы можете изменить любые настройки под свои потребности:
- Добавить новые сниппеты в `snippets.json`
- Расширить правила в `rules`
- Создать новые промпты в `prompts`
- Настроить параметры AI в `settings.json`
